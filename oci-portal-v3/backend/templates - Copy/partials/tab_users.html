Role permissions
Admin
Full access. Manages users, all compartments, all regions.
  • All instances, all regions
  • Start / Stop / Reboot
  • All compartments
  • Manage users & tags
  • Full audit + debug
Operator
Configurable: scope, actions, and OCI tag rules applied per user.
  • Scoped compartments only
  • Permitted actions only
  • Tag-matched instances only
  • Cannot manage users
  • Own audit entries
Viewer
Read-only. Sees scoped instances, no actions.
  • Scoped compartments only
  • No Start / Stop / Reboot
  • Tag-matched instances only
  • Cannot manage users
  • No audit access
